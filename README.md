Create Project
====================
```
mkdir project-name
```

```
cd project-name
```

```
touch README.md
touch demo.js
touch .gitignore
```

Edit demo.js
=====================
```
npm init -y
npm install
```

Add file to gitignore
=====================
```
echo "filme name" >> .gitignore
```

Create repository and push
=====================
```
git init
hub create (tame)
git add
git commit -m
git push origin master
hub browse
```
